Beside node.js and io.js, I'm gonna add vertex as a new platform. The sample of JS application using vertex is a follow: 

var vertx = require('vertx');
vertx.createHttpServer().requestHandler(function(req) {
	var file = req.path() === '/' ?
		'index.html' : req.path();
	req.response.sendFile
		('webroot/' + file);
}).listen(8080);

It looks easier than node.js and io.js, it seems to run faster than the current node.js/io.js platform.
