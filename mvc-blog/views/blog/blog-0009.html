Big Improvement Inheritance and Polymorphism

2015-04-04

The Cool! programming language is now support inheritance by "extends" keyword similar to Java. Currently it supports only single inheritance like in Java / C#, but it is possible to do multiple inheritance too.

Inheritance in Cool! is the same thing as in JavaScript which is not like other languages. A variable b of class B extends from A is an intance of B only , it is not an instance of A.

Polymorphism is also in the JavaScript way, the new member function will replace the existing identical name.
