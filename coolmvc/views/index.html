The Cool! programming language is a small computer language for server and browser. It comes with small framework so you can learn the language and framework in just an hour.

The Cool! Idea

  • SAB - Server And Browser - Use only one single language from server to browser
  • TOI - Text editor Over IDE - You can use any text editor to write Cool!
  • LIS - Life Is Short - Don't spend 10000 hours learning large framework.
  • PYS - Powerful Yet Small - Only 20 methods, but produce the same result as any large framework.
  • LOE - Learn Online Easily - No installation required. Cool! has an online editor.

Model View Controller (MVC)

MVC Framework in Cool! is simple and memorable. You can code without any IDE or generator. This framework is not only for Cool! but it can run on Node.js and io.js using server-side JavaScript and ES6.

There is no popular MVC framework in Node.js yet, you can use this framework with your existing Node.js code. This framework designed to code from any text editor easily. With just several methods in a few classes such as Controller, View, Database and Server. So you can learn Cool! MVC framework within an hour.

MVC Sample

main class Simple extends Controller {
	new {
		var server = new Server(this);
		server.start();
	}

	method index(context) {
		context.response.end("Cool!");
	}
}

Server And Browser

Cool! can use on both server and browser. You can use by writing Cool! code inside HTML or separate Cool! code file. Don't forget to include the cool.js file in your HTML file.
