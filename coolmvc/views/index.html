The Cool! Programming Language

Cool! Ideas

  • SAB - Server And Browser - Use only one language from server to browser
  • TOI - Text editor Over IDE - You can use any text editor to write Cool!
  • LIS - Life Is Short - Don't spend 10000 hours learning large framework.
  • LOE - Learn Online Easily - No installation required. Cool! has online editor.

Please visit our GitHub for more information.

Model View Controller (MVC)

MVC Framework in Cool! is simple, compact and memorable. You can code without any IDE or generator. You can also use Cool! MVC with Node.js and io.js.

MVC Sample

main class Simple extends Controller {
	new {
		var server = new Server(this);
		server.start();
	}

	method index(context) {
		context.response.end("Cool!");
	}
}

Server And Browser

Cool! can use on both server and browser. You can use by writing Cool! code inside HTML or separate Cool! code file. Don't forget to include the cool.js file in your HTML file.
<script src="/cool.js"></script>

Browser Sample

<script type="text/cool" src="code.cool"></script>
			
<script type="text/cool">
... Your Cool! code here ...
</script>

Transpile to JavaScript Directly

Cool! can transpile to JavaScript and ES6 directly. You can try writing Cool! code in the left editor and it will transpile immediately to the right editor.
