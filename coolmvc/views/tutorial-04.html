Using Cool! MVC with JavaScript and ES6

You can also use Cool! MVC with JavaScript or ES6. The following code is an example of JavaScript server (Node.js or io.js). 

function Simple () {
	Controller.call(this);

	this.index = function (context) {
		context.response.end("Cool!");
	}
}

var simple = new Simple();
var server = new Server(simple);
server.port = 2003;
server.start();
